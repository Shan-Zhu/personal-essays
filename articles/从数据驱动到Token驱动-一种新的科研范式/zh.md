# 从数据驱动到Token驱动，一种新的科研范式

（1）

科学研究的范式，也许，正在经历一次新的转变。

过去二十年，"数据驱动"是科研方法论的主旋律。

建造大型强子对撞机来产生数据，发射空间望远镜来采集数据，构建基因测序流水线来积累数据。核心逻辑很清晰，世界蕴含着模式，数据是模式的载体，算法是提取模式的工具。

但一种新的范式正在悄然成型——**Token驱动范式**。

它的核心逻辑与数据驱动有着微妙而深刻的不同。

新知识的产生，不再必须从对自然的直接测量开始，而是可以从**人类已有知识的符号化表征（Token）的重新组合、压缩与推理**中涌现。

这不是一个修辞上的区分。

AlphaFold预测蛋白质结构，不是从新的实验数据中学习，而是从已有的序列-结构对（本质上是token化的生物学知识）中学习。

DeepMind的GNoME从已有晶体结构的token化表示中预测了220万种新稳定材料。

药物发现领域，分子的SMILES表示就是一种token化，生成式模型在token空间中"发明"新分子。数学领域，LLM辅助证明本质上是在token空间里搜索证明路径，不需要任何"数据"。

这些案例有共同点，**新知识的产生发生在token空间内部，而非从新的物理测量中涌现。**

如果说数据驱动的核心操作是"从自然中提取模式"，那么Token驱动的核心操作是"**从人类已有的全部知识中推理出尚未被写下的部分"**。

这个转变看似只是方法论的迭代，但当我们逐层深入，会发现它触及了科学最深处的几个根本问题：知识的本质是什么？理解意味着什么？科学的边界在哪里？

（2）

科学的最高成就本来就是Token操作。

回顾科学史，许多最深刻的理论突破，本质上就是在符号空间中的操作。

狄拉克预测反物质，不是从实验数据中发现的，而是因为他的方程有一个负能量解，他拒绝丢弃一个数学符号。

麦克斯韦加入位移电流项时没有任何实验依据，纯粹是因为方程的对称性在符号层面不完整。

爱因斯坦构建广义相对论，核心工具是思想实验加黎曼几何的符号体系，实验验证是事后的事情。

这些都是纯粹的符号结构中的空位。

这意味着什么？

**理论物理学的最高成就，本质上就是在数学符号系统，或者说**某种token空间，**做搜索和补全。**

所以，当我们说科研范式正在"转向Token驱动"时，这个表述需要被修正。更准确的说法是：

Token驱动不是一种全新的范式，而是科学终于意识到了自己一直在做的事情的真正本质。

大语言模型没有发明Token操作这种科研方法，**它让这种方法的本质变得可见了。**

过去，只有极少数天才（狄拉克、爱因斯坦他们）能在符号空间中进行高效搜索。现在，这种能力正在被工业化。

（3）

如果token驱动只是让已有的科研方法变得更高效，那它的意义是有限的。

但它的影响远不止于此，它正在重塑科学发现的权力结构。

数据驱动范式有一个隐藏的前提，**数据是稀缺的、昂贵的、且往往被少数机构垄断的。**

你需要CERN的对撞机、哈勃望远镜、几百万美元的基因测序设备，或者在顶级实验室才能接触到的独家数据集。数据驱动的本质是资本密集型科学，它天然地把科学发现的权力集中在少数机构手中。

Token驱动范式部分地打破了这个结构。当知识的原材料从"独占的数据"变成"公开的token化知识"时，一个在内罗毕的研究者和一个在MIT的研究者，第一次面对几乎相同的原始知识空间。

但这里有一个更暗的推论不应被忽视。如果Token驱动成为主导范式，**算力就成为新的对撞机**。

科学的权力结构不会真正民主化，只会从"拥有数据的人"转移到"拥有算力的人"。

从CERN到NVIDIA，从实验室到数据中心，机构变了，但权力的集中性可能不变，甚至加剧。

范式转移从来不单单是科学的事，同时也是政治与经济的事。

（4）

还可以进行深入想想。

传统科学有一个隐含的承诺，**我们不只是要预测，我们要理解。**

牛顿力学不只是告诉你苹果会落地，它告诉你为什么。"理解"，这事是以简洁的因果叙事压缩复杂现象，它一直被视为科学的核心价值。

但Token驱动范式暗示了一种新的可能性：

也许"理解"只是人类认知带宽有限时的一种必需品，它是一种压缩策略。当你有足够的Token处理能力时，你不需要"理解"，你只需要在Token空间中找到正确的路径。

人类需要"理解"，是因为我们的工作记忆只有7±2个组块，必须把复杂现象压缩成简洁的定律才能操作。

一个万亿参数的模型不需要这种压缩，它可以直接在高维Token空间中导航到正确答案，走一条人类永远无法"理解"的路径。

AlphaFold预测蛋白质结构，但没有生物学家能完整解释它的推理过程。AlphaGo的第37手，人类棋手花了数月才勉强理解为什么它是对的。数学中已经出现了计算机辅助证明，正确性可验证，但没有人类能"读懂"全部推理。

如果这个趋势继续，科学可能分裂成两种活动：

**一种是token空间中的搜索**——高效、正确、但不透明。

**一种是人类可理解的叙事建构**——低效、可能有误、但满足人类对"意义"的需求。

那么，第二种活动会不会退化成一种事后合理化，就像我们对自己的直觉决策编造理由一样？如果是这样，"科学理解"这个概念本身可能需要被重新定义。

（5）

Token驱动范式的力量是巨大的，但它有一个根本性的极限。

如果我们把"全人类已有的科学知识"看作一个形式系统，那么是否**存在命题，无法从现有Token体系内部推导出来。**

**换句话说，存在一些真理，其概念根本不在我们现有的Token词表中。**

假设你是1850年的一个超���AI，你拥有当时全人类所有物理学知识的Token化表示，你能推导出量子力学吗？

不能。因为"波函数"、"叠加态"、"不确定性原理"这些概念在1850年的Token空间中不存在，它们不是现有概念的组合。普朗克引入量子化假说时，他不是在已有的Token空间中搜索，他是创造了一个新Token。

因此，Token驱动范式面对的边界至少有三类：

**第一类是组合性空白。** "我们知道A，知道B，但从没有人把A和B放在一起考虑过"，例如把拓扑学应用于凝聚态物理。所有需要的Token都已存在，只是从未被正确组合。Token驱动系统在这里极其强大，因为它天然地能看到跨领域Token共现结构中的空白。

**第二类是表征性空白。** "我们现有的概念体系中存在一个系统性的盲区，但我们甚至不知道缺失的概念应该长什么样"，就像1850年的物理学缺少"场"的概念。Token驱动系统可以部分检测到这类空白，模型的隐空间可能包含一些没有对应任何人类已命名概念的维度，这些"无名维度"是新概念的模糊投影，但它无法用现有token清晰地描述它们。

**第三类是框架性空白。** 不是缺少某个概念，而是我们组织概念的整个方式就有局限。从牛顿力学到量子力学，改变的不只是几个概念，而是"物理理论应该是什么样子"这个框架本身。Token驱动系统在这里遭遇硬边界，因为它的框架，是被架构和训练过程预先决定的。正如维特根斯坦所说："我的语言的界限，意味着我的世界的界限"。

（6）

Token驱动范式的最深远意义，不在于它能在已知空间中找到多少新答案虽然这已经是一个巨大的宝藏，它的最深远意义在于一个前所未有的能力：

**系统性地、定量地、大规模地测绘人类认知的边界。**

这种测绘如何可能？关键机制是，**从缺席中读出信息。**

一个在海量科学文献Token上训练的模型，内部形成了一个巨大的知识流形。这个流形不只记录了"什么是已知的"，它隐式地编码了"已知事物之间的结构关系"。当这个结构中出现不自然的空洞、扭曲或不一致时，这些拓扑异常本身就是信号。

想象这样一个场景，你让一个模型在某个特定条件下做预测，它的输出突然变得高度不确定、自相矛盾，或者输出分布出现异常的多模态性。这种"困惑"不是模型的失败，而是它标记出了知识结构中的断裂带。

这是一个认识论的倒转：传统科学中，知识是信号，无知是噪声。在Token驱动的边界测绘中，**无知本身变成了信号。**

人类科学史上最深刻的发现，往往始于注意到一个结构性的缺席。

门捷列夫看到的不是已知元素，而是周期表中的空格。泡利看到的是β衰变中缺失的能量。天文学家看到的是星系旋转曲线中缺失的质量。但这些发现是零星的、偶然的、依赖个体天才的直觉的。

Token驱动系统第一次让这种"缺席检测"变得系统化、工程化。

（7）

如果上述机制成立，Token驱动范式将产出一个人类历史上前所未有的元科学对象：

一个关于"人类目前不知道什么"的知识库。

我们有过知识的百科全书，从狄德罗到维基百科，但我们从未拥有过无知的百科全书。

这个无知之书的存在将产生深远的后果。

**首先，它将改变科研资源的分配逻辑。**目前科研资源的分配很大程度上由社会学因素驱动，哪个领域热门、哪个PI有影响力、哪个方向容易发论文。而一张无知地图将提供一个前所未有的客观参照，这里有一个巨大的认知空洞，周围的知识结构暗示这里可能存在重要的未知，但目前几乎没有人在研究它。科研资助机构第一次有可能基于知识结构本身的拓扑特征来分配资源，而不是基于同行评议。

**其次，它将重新定义"什么算作重要的科学问题"。** 目前，"重要问题"的定义由学科传统、领域精英或者重要人物决定。而这本Token揭示的无知之书会，不能告诉你这些问题是什么，因为它们超出了Token空间，但它能告诉你这些问题的投射在已知知识结构上的位置。就像天文学家通过引力透镜效应来推断暗物质的分布，你看不见暗物质，但你能看到它对可见光的扭曲。

**最后，也是最深刻的，它可能改变科学对自身使命的理解。** 科学的传统自我叙事是，我们在逐步揭示自然的真理，未知在不断缩小。但如果无知地图告诉我们的是，我们知道得越多，我们能测绘到的"未知边界"就越长，而不是越短？

知识的边界是一个分形结构，分辨率越高，边界线越长。

科学不是一个逐渐填满的容器，它是一个不断扩张的球体，体积在增长，但表面积（与未知的接触面）增长得更快。Token驱动系统第一次让我们能定量地看到这个表面积的增长速度。

这会迫使科学重新理解自己：

我们有机会，科学地将"不知道自己不知道"转化为"知道自己不知道"。

（8）

Token驱动范式的兴起，表面上看是科研工具的一次升级，从用数据训练模型，到用Token化的知识进行推理。但当我们逐层深入，会发现它触及的东西远比工具层面深刻。

但同时，这里存在一个根本性的极限。存在一些真理，需要的概念根本不在我们现有的Token词表中，任何Token内部的操作都无法触及它们。

而恰恰是这个极限，指向了Token驱动范式最深远的贡献：

它不会替我们跨越认知的边界，但它会第一次让我们清楚地看见边界在哪里，边界的形状是什么。

边界的另一边，隐约有什么在等着我们。
